音声ファイルの作り方（Windows） ›
  1. Windows の「設定」→「システム」→「サウンド」→「すべてのサウンドデバイス」を開く
  2. 入力デバイスの一覧から ステレオミキサー を選択し「有効にする」をクリック

    ※ ステレオミキサーとは、PC が出力している音をそのまま録音できる仮想入力デバイスです。マイク配列は外部マイクからの音を拾うもので、用途が異なります。

  3. Windows の「サウンドレコーダー」アプリを起動する
  4. 画面左下の入力デバイスを「ステレオミキサー」に変更する
  5. サウンドレコーダーで録音を開始する
  6. このアプリでメトロノームを再生する
  7. メトロノームが終了したら、サウンドレコーダーで録音を停止する
  1. Open Windows Settings → System → Sound → All Sound Devices
  2. Under input devices, select Stereo Mix and click Enable

    * Stereo Mix is a virtual input that captures audio playing on your PC. Microphone Array captures sounds from an external mic — they serve different purposes.

  3. Open the Windows Sound Recorder app
  4. Change the input device (bottom-left) to Stereo Mix
  5. Start recording in Sound Recorder
  6. Play the metronome in this app
  7. Stop recording in Sound Recorder when the metronome finishes

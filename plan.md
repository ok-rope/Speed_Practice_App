# なわとびスピード練習メトロノーム - 提案書

## 概要

なわとびのスピード競技（特に30秒スピード）向けに、時間帯ごとにBPMを変えられるメトロノームWebアプリ。
実際の競技ペースを忠実に再現した練習音源を生成し、MP3としてダウンロードできる。
スマートフォン・PCどちらでも快適に使用可能。

---

## BPMの考え方

| なわとびの回数（回/分） | メトロノームBPM |
|---|---|
| 80回 | 160 BPM |
| 79回 | 158 BPM |
| 78回 | 156 BPM |
| 76回 | 152 BPM |

**計算式：BPM = 回数 × 2**

---

## 機能要件

### 1. 全体時間の設定
- 練習時間を自由に設定（デフォルト：30秒）
- 入力範囲：1秒〜300秒

### 2. セグメント設定（タイムライン）
- 全体時間を任意の区間（セグメント）に分割できる
- セグメント数に制限なし（最低1つ）
- 各セグメントに以下を設定：
  - 開始時間（秒）
  - 終了時間（秒）
  - なわとびの回数（回/分）→ 内部でBPM変換
  - モード：**ステップ（即時切替）** または **グラデーション（徐々に変化）**
- セグメントの追加・削除が可能

**グラデーションモードの動作：**
- セグメント内で開始BPMから次セグメントのBPMへ線形補間
- 各ビートごとに次ビートまでの間隔を再計算してスケジューリング

**設定例（30秒スピード）：**
```
0〜10秒:  80回/分 → BPM 160（ステップ）
10〜21秒: 79回/分 → BPM 158（ステップ）
21〜25秒: 78回/分 → BPM 156（ステップ）
25〜30秒: 76回/分 → BPM 152（ステップ）
```

### 3. アナウンス機能

**開始アナウンス（自由記述）：**
- 再生前に読み上げるテキストを自由入力
- 例：「Single Rope Speed Sprint 30 seconds」
- TTS（ブラウザの音声合成）で読み上げ完了後にカウントダウンへ移行

**カウントダウン：**
- 0〜10秒の整数で指定（デフォルト3秒）
- 例：3秒 → "Three, Two, One" と読み上げ → スタートBEEP → メトロノーム開始

**回数アナウンス：**
- セグメント開始・切り替わり時に回数を英語で読み上げ
- 例：「eighty」「seventy six」

**時間アナウンス：**
- 任意の時間（秒）を複数指定
- その時刻になったら経過秒数を英語で読み上げ

### 4. BEEP音
- **スタートBEEP**：最初のビートの合図（メトロノーム開始の瞬間）
- **切替BEEP**：セグメント切り替わりの合図
- **終了BEEP**：メトロノーム終了の合図
- 各BEEPは独立してON/OFF可能

### 5. 機能ON/OFFトグル（7項目）
| トグル | 内容 |
|---|---|
| 開始アナウンス読み上げ | 自由記述テキストのTTS読み上げ |
| カウントダウン音声 | "Three, Two, One" のTTS読み上げ |
| スタートBEEP | 最初のビートのBEEP音 |
| 切替BEEP | セグメント切り替わりのBEEP音 |
| 終了BEEP | メトロノーム終了のBEEP音 |
| 回数アナウンス | 回数の英語読み上げ |
| 時間アナウンス | 指定時刻の英語読み上げ |

### 6. クリック音の種類
- 電子音（デフォルト）
- 木琴風
- シンプル

### 7. リアルタイムプレビュー再生
- ブラウザ上でそのまま再生
- タイムラインバー上に再生位置インジケーターをリアルタイム表示
- 現在の回数・残り時間・セグメント番号を表示

### 8. タイムライン（統合表示）
- 色付きセグメントバーとBPM形状ラインを1つに統合
- セグメントバーの色は回数に応じてHSLカラーで変化
- 白いライン（ステップ：水平線＋垂直線、グラデーション：斜め線）とドットをバー上にオーバーレイ表示

### 9. MP3ダウンロード
- ブラウザ内でOfflineAudioContextを使用してオフラインレンダリング
- lamejs（CDN）でMP3エンコード（128kbps）
- カウントダウン区間は無音（TTS音声は埋め込み不可）

## UI設計（スマホ対応・レスポンシブ）

縦1列レイアウト、ボタン大型化（最小高さ52px）、タッチ操作最適化。
最大幅680px、中央寄せ。スマホ・デスクトップで共通レイアウト。

## 技術スタック

| 技術 | 用途 |
|---|---|
| HTML / CSS / JavaScript | ベース（ビルドツール不要） |
| Web Audio API | リアルタイムメトロノーム再生・音声生成 |
| Web Speech API | TTS（アナウンス・カウントダウン） |
| OfflineAudioContext | MP3用オフラインレンダリング |
| Canvas 2D API | タイムラインBPM形状ライン描画 |
| lamejs（CDN） | MP3エンコード（ブラウザ内） |
| CSS Grid / Flexbox | レスポンシブレイアウト |
